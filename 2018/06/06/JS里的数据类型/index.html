JS里的数据类型

Posted on 2018-06-06

概述

JS有7种数据类型：number string boolean symbol undefined null object

number

整数和小数：1 1.1 .1
科学记数法：1.23e2
二进制：0b11
八进制：011（后来 ES5 添加了 0o11 语法）
十六进制：0x11

string

空字符串：’’
多行字符串：

boolean

true
false

symbol

Symbol 是EC6规格所支持的一种新的数据类型
Symbol 生成一个全局唯一的值
Symbol 可以创建一个独一无二的值（但并不是字符串）

undefined 和 null

变量没有被赋值–undefined
有一个对象，不想赋值，则初始化成–null
有一个非对象，不想赋值则初始化成–undefined

object

object 的 key 一律是字符串，不存在其他类型的 key
object[‘’] 是合法的
object[‘key’] 可以写作 object.key
注意 object.key 与 object[key] 不同
delete object[‘key’]
‘key’ in object
object 里面可以有 object
