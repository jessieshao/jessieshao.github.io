TCP传输控制协议(Transmission Control Protocal)

1.TCP和UDP的区别是什么？
TCP可靠、面向连接、相对较慢
UDP不可靠、不面向连接、相对较快
2.TCP的三次握手是什么？
（1）客户端：我要连接你了，可以吗？
（2）服务端：嗯，我准备好了，连接我吧
（3）客户端：那我连接你咯

IP网络协议(Internet Protocol)

localhost 127.0.0.1
路由器连接内网和外网
内网ip一般为c类地址192.168.x.x

端口

要访问一个设备，除了指定ip还要指定端口(port)
端口就是一个编号
一个端口对应一个服务
一个服务器不一定只提供一个服务：HTTP服务80端口、FTP服务21端口、SMTP服务
一共有65535个端口，1023以下是留给系统用的
