git入门

Posted on 2018-04-19

git init

  • 在我们新建一个空仓库后需要初始化这个仓库
    1.首先新建一个项目目录demo
    2.命令行进入demo下
    3.git init在demo中创建一个.git目录，她就是一个仓库

git add

  • 此时在demo下创建一个文件后，只是创建成功，并没有保存
    1.touch 1.txt创建一个txt文件
    2.git status -sb查看状态，发现新建文件前为？
    3.此时用git add 1.txt添加进暂缓区
    4.再次git status -sb发现已存入

git commit -v

  • 使用git commit -m“信息” 将你 add 过的内容「正式提交」到本地仓库，并添加一些注释信息，方便日后查阅
    1.git commit -v 提交时显示所有diff信息
    2.完成后才是正式提交上
